🗽 City Block Combined Prototype

An NYC-inspired multi-camera pedestrian capture and visualization system.

Project Outline

  • 🧠 AI pedestrian tracking (YOLOv8 + MOT)
  • 🎥 30+ camera layout across NYC-style block
  • 🌐 Viewers: Unity, Blender, Three.js
  • 🌀 Gaussian splats, contrails, edge maps
  • 👣 Interactive walkthrough (VR-ready)
  • 🧍 NPC JSON defining live pedestrian paths
